### 주요 기능


#### 타이머

집중 모드: 25분 (기본값)<br>
짧은 휴식: 5분 (기본값)<br>
긴 휴식: 15분 (기본값)<br>
시간 커스터마이징 가능<br>
일시정지/재시작 지원<br>
백그라운드에서도 타이머 유지<br>

<br>

#### 테마

라이트 모드<br>
다크 모드<br>
고양이 테마 <br>
안경 고양이 테마

<br>

#### 다국어 지원

한국어 (기본)<br>
English<br>
日本語<br>
브라우저 언어 자동 감지

<br>

#### 알람 옵션

기본<br>
벨<br>
3회 알람<br>
멜로디<br>
마림바<br>
띵동

<br>

#### 통계

완료한 뽀모도로 개수 추적<br>
총 집중 시간 계산 (분)<br>
통계 초기화 기능

<br>

### 추가 기능

타이머 이름 커스터마이징<br>
브라우저 알림 지원<br>
키보드 단축키
- Space: 시작/일시정지
- R: 리셋


LocalStorage를 통한 설정 저장<br>
반응형 디자인 (모바일 지원)

<br>

### 사용 방법

1. https://zada8.github.io/pomodoro/ 브라우저에서 열기
2. 우측 상단 "설정" 버튼으로 원하는 테마, 시간, 알람음 선택
3. 모드 선택 (집중/짧은 휴식/긴 휴식)
4. "시작" 버튼 클릭 또는 Space 키로 타이머 시작

<br>
